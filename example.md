# Welcome to nvim-slides

---

## Getting Started

- Install the plugin
- Open a markdown file
- Run `:SlidesStart`

---

# **Bold** Title Slide

---

## Features

- Markdown-based slides
- **Bold text** support
- Nested bullets:
  - Second level
    - Third level
- Simple navigation

---

## Navigation

- Press `n` or `l` to go **forward**
- Press `p` or `h` to go **back**
- Press `q` to **quit**
