# Slides

Make presentations in markdown and present in Neovim I guess.

## Motivation

idk, I was about to give a presentation and felt like using my markdown notes
to do it.

## Installation

```lua
vim.pack.add("https://github.com/ptdewey/slides-nvim")
```

## Usage

See [example.md](example.md) for syntax details, it should be fairly common-sense.
Use the `SlidesStart` command to go into presentation mode.
